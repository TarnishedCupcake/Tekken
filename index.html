4. Stance System & Transitions

This section breaks down Steve’s stances including inputs, transitions, key stance-specific moves with frame data, and traits like Homing and Power Crush.

Peekaboo (PAB)

  • Entry: F+3+4, 2,1F
  • Transitions: F+3 → DCK, B → Sway
  • Core Moves:
    • PAB 1: 15f / -5 / +3 / +5
    • PAB 2: 18f / -8 / +6 / +8
    • PAB 1+2 (Throw): 12f / Break / Knockdown / Throw

Flicker (FLK)

  • Entry: B+3+4, 2,1B, B+1 (on hit)
  • Transitions: F → ALB, B → Sway
  • Core Moves:
    • FLK 1: 11f / -3 / +6 / +8
    • FLK 2: 13f / -4 / +3 / +6
    • FLK 1+2: 19f / -8 / KND / KND 🔥 Homing

Ducking (DCK)

  • Entry: F+3~F, PAB → F+3~F
  • Core Moves:
    • DCK 1: 14f / -7 / +2 / +4
    • DCK 1+2: 20f / -9 / KND / KND

Sway (SWY)

  • Entry: B+3 / B+4 (from FLK or PAB)
  • Use: Evade highs/mids, cancel out of pressure
  • Core: Cancel into jab/B+1

Left/Right Weave

  • Left Weave (3): Sidestep left, evasive tool
  • Right Weave (4): Sidestep right, combo adjuster

Albatross (ALB)

  • Entry: 3+4 or FLK → F
  • Core Moves:
    • ALB 1: 22f / -10 / +4 / KND
    • ALB 2: 24f / -9 / +6 / KND 🔥 Power Crush

Lionheart

  • Input: U/B+3
  • Use: Evasive launcher tool, momentum shift
  • Frames: 15f / -10 / +3 / +7

6. Gameplan and Setups

This section is centered around Steve’s strongest neutral starters and cancel pressure routes. Emphasis is placed on flow control, transitions from key pokes, stance entry, pressure loops, and fuzzy-breaking resets.

Core Neutral Starters

  • 1, 1,2,1: Safe jail string, easy hit confirm
  • D/F+1: Safe mid poke, leads to jab pressure
  • 2,1B: Transitions to FLK, starts cancel chain
  • 2,1F: Transitions to PAB, frame trap opener

Cancel Loops & Pressure Chains

  • 2,1B → FLK 1,1,F1 → DCK cancel → FLK reset
  • 1 → 1 → Sway cancel → jab trap → FLK 2
  • CH B+1 → FLK 2 → Delay → PAB 2 → DCK 1

Keep-Out and Defensive Tools

  • B+2: Homing move to stop sidesteps 🌀
  • F,F+2: Long-range whiff punish 🔥
  • D/B+2: Low crush and oki knockdown

Pokes & Small Tekken

  • 2,1B → FLK jab loop: Microspace pressure
  • D+1: Safe low, resets pace
  • 1, delay 1: Check fuzzy/interrupt
  • FLK 1 → FLK 2 → Backdash → FLK 1+2

Frame Trap Layers

  • FLK 1 → Delay FLK 1+2: Mid trap reset
  • 2,1B → CH FLK 1 → ALB 2
  • D/F+1 → 2,1F → PAB 2 → DCK delay

Okizeme

  • D/B+2 oki: Covers wakeup and roll
  • FLK cancel → jab → DCK 1: Ground tech trap
  • Wall hit → walk jab reset → FLK cancel

Flowcharts

  • 1,2,1 → FLK → jab reset or delay FLK 2
  • 2,1F → PAB → DCK 1 or throw
  • D/F+1 → jab → 2,1B → FLK loop

Round Starters

  • D/F+1 (safe poke)
  • 1 → walk → jab (frame buffer bait)
  • B+1 (CH gamble setup)

Round Closers

  • FLK 1 → throw (mental crush)
  • D+1 or D/B+1 (chip poke)
  • FLK cancel → jab → low (stagger closer)

Timer Strategy

  • 40s+: Play keepout with B+2, D/F+1
  • 20s–10s: Activate Heat or chase wall
  • Under 10s: Use FLK delay strings or throw/low reset

  • FLK 1 → DCK 1 → Heat Dash → Ender

  • Build 10-card matchup deck (punish + weak side)
  • Watch 5 replays vs Steve or vs hard counters
  • Lab FLK parry break options

Week 6: Wall Tech & Reset Study

  • W! → walk jab → FLK reset or PAB throw
  • Fake FLK cancel into throw/low vs tech roll
  • F,F+2 → DCK 1 → Heat TOD at wall

Week 7: Panic Bait and Defensive Stack

  • Delay FLK 1 → CH bait → confirm drill

9. Matchups

This section focuses on high-level knowledge for fighting common tournament characters. Each profile includes weak sidestep direction, punish notes, and Steve-specific strategy tips.

Jin Kazama

  • Weak Side: Sidestep Left
  • Punish: 2,4 (-10), F,F+2 (-13)
  • Strategy: Use B+1 to counter jab chains. FLK cancel trap on parry attempts.

Kazuya Mishima

  • Weak Side: Sidestep Left
  • Punish: Hellsweep (-17), EWGF (-10)
  • Strategy: Whiff punish electric with F,F+2 or FLK jab trap.

Leroy Smith

  • Weak Side: Slight Left
  • Punish: Hermit 1+2 (-14), B+1+2 parry (-10)
  • Strategy: Delay pressure to bait sabaki. Use FLK 2 over 1s.

Marshall Law

  • Weak Side: Sidestep Right
  • Punish: DSS 2 (-10), DSS 1+2 (-14)
  • Strategy: Backdash → FLK cancel → check DSS auto-pilot.

Reina

  • Weak Side: Sidestep Left
  • Punish: Demon Slash (-12), F+2,3 (-10)
  • Strategy: Stay grounded vs teleport. Use FLK jab to test panic.

Paul Phoenix

  • Weak Side: Sidestep Left
  • Punish: Deathfist (-12), D+1 (-13)
  • Strategy: Use FLK 1 to space out CH attempts. Sway cancel loop baits.

Hwoarang

  • Weak Side: Sidestep Right
  • Punish: RFF D/B+4 (-13), LFS 3 (-10)
  • Strategy: Interrupt stance entry with jabs. B+1 to CH step cancels.

Jun Kazama

  • Weak Side: Sidestep Right
  • Punish: D+3,2 (-13), F+1+2 (-10)
  • Strategy: Walk away from parry windows. Use low crush D/B+2 on read.

Alisa Bosconovitch

  • Weak Side: Sidestep Left
  • Punish: B+3,3 (-14), D/B+3 (-13)
  • Strategy: Step right chainsaws. Use F,F+2 on whiff attempt.

King

  • Weak Side: Sidestep Left
  • Punish: F+1+2 (-12), SHD 1+2 (-14)
  • Strategy: Delay string to bait throw mix. Backdash out of chain throw range.

10. Anti-Steve Tech

This section explores how experienced opponents counter Steve’s stance system and how to adapt to reverse these counters effectively.

Common Steve Counters

  • High Crush: Duck under FLK 1 or jab strings
  • Sidewalk Right: Evades FLK 2, FLK 1+2
  • Low Parry: Common vs D+1 or Ducking lows
  • Throw Tech: PAB 1+2 is predictable when overused

Parry and Sabaki Risk

  • Jun’s Parry: Hits FLK 2, PAB 1
  • Leroy Hermit Sabaki: Breaks FLK 1+2 strings
  • Asuka’s Reversal: Stops linear cancel attempts

Stance Break Chart

StanceOpponent CounterHow to Beat It
FLKSidewalk RightCancel → DCK 1 or throw
PABJab interruptDelay → B+1 (CH trap)
DCKLow parryCancel → FLK 2 → FLK 1+2
SwayBackdash or waitEmpty cancel → jab → trap

Counter-Adaptive Strategy

  • If opponent sidewalk FLK: Cancel → DCK 1+2 or FLK 1
  • If opponent fuzzy guards PAB: Delay jab into throw/low
  • If opponent low parries DCK: Cancel → jab → sway reset
  • If opponent parries FLK: Use CH jab loop → backdash FLK 2

11. Controller-Specific Tips

Optimize Steve’s execution-heavy stance transitions, cancels, and Heat usage based on your control setup (pad or stick).

For Pad Players

  • Bind Suggestions: 1+2 and 3+4 on shoulder buttons
  • Heat Activation: Map Heat Burst to L2/R2 for smoother access
  • Execution Tip: Hold FLK with thumb, DCK input with index flick
  • Practice Drill: 2,1B → FLK 1+2 → DCK 1 (Heat Dash confirm)

For Stick Players

  • Layout Advice: Place 1+2, 3+4 near main attack row
  • Wrist Roll: Use for DCK cancels, Heat dash execution
  • Practice Drill: B+1 → FLK 2,1 → DCK 1+2 → Heat Dash → Ender

General Tips

  • Jab Confirm Buffer: Practice FLK 1 → delay → FLK 1+2
  • Option Selects: PAB throw/low loops via macro timing
  • Visual Training: Practice input overlays with replay tools
