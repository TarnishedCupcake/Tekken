4. Stance System & Transitions

This section includes all of Steve’s stances with frame data, transitions, and tactical notes. Each stance-specific move includes startup, on hit/block, and CH properties.

Peekaboo (PAB)

  • Entry: F+3+4, 2,1F
  • Transitions: F+3 → DCK, B → Sway
  • Key Moves:
    • PAB 1 — 15f / -5 / +3 / +5
    • PAB 2 — 18f / -8 / +6 / +8
    • PAB 1+2 — 12f / Break / Throw

Flicker (FLK)

  • Entry: B+3+4, 2,1B, B+1 on hit
  • Transitions: F → ALB, B → Sway
  • Key Moves:
    • FLK 1 — 11f / -3 / +6 / +8
    • FLK 2 — 13f / -4 / +3 / +6
    • FLK 1+2 — 19f / -8 / KND / KND 🔥 🌀

Ducking (DCK)

  • Entry: F+3~F, PAB F+3~F
  • Key Moves:
    • DCK 1 — 14f / -7 / +2 / +4
    • DCK 1+2 — 20f / -9 / KND / KND 🔥

Sway (SWY)

  • Entry: B+3 / B+4 (from FLK/PAB)
  • Function: Evade highs/mids, cancel to reset spacing

Left/Right Weave

  • Left Weave (3) — Evasive sidestep left
  • Right Weave (4) — Evasive sidestep right

Albatross (ALB)

  • Entry: 3+4 or FLK → F
  • Key Moves:
    • ALB 1 — 22f / -10 / +4 / KND
    • ALB 2 — 24f / -9 / +6 / KND 🔥

Lionheart

  • Entry: U/B+3
  • Function: High-evasion launch, counterplay to aggression
  • Frames: 15f / -10 / +3 / +7

6. Gameplan and Setups

This section outlines Steve’s offensive and defensive philosophy, stance pressure, pokes, and layered mindgames. Cancel pressure, frame traps, and flowcharts are fully integrated.

Aggressive Playstyle

  • Use 1, 1,2,1 to test opponents' mash defense
  • 2,1B → FLK jab pressure → DCK cancel loop
  • CH B+1 → FLK 2,1 → force wall carry or reset
  • FLK 1 → FLK 1+2 → backdash → FLK jab loop

Defensive Playstyle

  • D/F+1, jab, 2,1 poke buffer
  • Backdash → F,F+2 whiff punish
  • Low crush with D/B+2 or parry baits
  • Use sway cancel to create space/reset stack

Neutral Starters

  • D/F+1, 1, 1,2,1
  • 2,1F (PAB), 2,1B (FLK)
  • F+2 into FLK reset

Keep-Out Moves

  • F,F+2 — long-range launcher
  • B+2 — homing sidestep check
  • D+1, D/B+1 — delay pokes

Pokes and Small Tekken

  • 1 → delay 1 → reset
  • 2,1 → check + cancel branch
  • D+1 → reset or fuzzy test
  • FLK 1 → jab → cancel → throw/low

Cancel Pressure Chains

  • 2,1B → FLK jab → FLK 1+2 or DCK 1
  • 1, walk → jab trap
  • FLK 1 → cancel → jab → throw

Frame Trap Routes

  • FLK 1 → delay FLK 1+2
  • 2,1F → PAB 2 → DCK delay
  • CH B+1B → FLK 2 → ALB 2

Okizeme

  • FLK 1 on grounded tech
  • D/B+2 on low wakeup
  • PAB cancel → delay → throw reset

Round Starters

  • 1, D/F+1, B+1
  • 2,1 → stance cancel check

Round Closers

  • FLK cancel → low/throw
  • FLK 1 → jab → DCK 1

Timer Strategy

  • 40s+ → play outside with F,F+2 / D/F+1
  • 20s → Heat or PAB throw loop
  • 10s → FLK feints into jab → cancel low

Flowcharts

  • 2,1F → PAB → throw or delay jab
  • FLK 1 → DCK 1+2 or backdash → FLK 1+2
  • CH B+1 → FLK 2,1 → DCK cancel trap
  • 2,1B → FLK jab → FLK 1,1 → FLK cancel

7. FoxStack Curriculum

This structured 8-week training cycle builds tournament-level Steve play through weekly lab goals, matchup research, and bracket simulations. Use with printable PDF tracker or GitHub markdown log.

Week 1: Fundamentals

  • Backdash cancel → jab spacing drill
  • 2,1B → FLK jab confirms (100 reps)
  • Punisher warmups (i10 to i14 only)

Week 2: Movement and Whiff Punish

  • Sidestep → B+1 or WS 1,2 (left/right reps)
  • Backdash → F,F+2 vs dummy (bait test)
  • Wall carry drill to FLK 1+2 reset

Week 3: CH Setups and Layered Pressure

  • B+1 → FLK 2 → PAB 1+2 trap loop
  • 2,1F → PAB 2 → DCK cancel delay trap
  • Reset route: D/F+1 → jab → FLK 1+2

Week 4: Heat Route Optimization

  • FLK 1 → DCK 1 → Heat Dash → Ender
  • Heat TOD practice vs CH B+1, CH F+2
  • F+2,1F → PAB → Heat Burst → Enders

Week 5: Matchup Notes & Anti-Steve Study

  • Build 10-card matchup deck (punish + weak side)
  • Watch 5 replays vs Steve or vs hard counters
  • Lab FLK parry break options

Week 6: Wall Tech & Reset Study

  • W! → walk jab → FLK reset or PAB throw
  • Fake FLK cancel into throw/low vs tech roll
  • F,F+2 → DCK 1 → Heat TOD at wall

Week 7: Panic Bait and Defensive Stack

  • Delay FLK 1 → CH bait → confirm drill
  • Practice DCK cancel → jab → backdash → FLK 2
  • FLK 1 → throw loop baiter (mental stack)

Week 8: Bracket Simulation & Tournament Prep

  • 20 ranked sets w/ notebook log
  • 15-minute matchup warmup + notes before pool
  • Print flowcharts + Heat TOD path checklist

  • Weak Side: Sidestep Left

  • Weak Side: Slight Left

  • Weak Side: Sidestep Right

  • Punish: Demon Slash (-12), F+2,3 (-10)
  • Strategy: Stay grounded vs teleport. Use FLK jab to test panic.

Paul Phoenix

  • Weak Side: Sidestep Left
  • Punish: Deathfist (-12), D+1 (-13)
  • Strategy: Use FLK 1 to space out CH attempts. Sway cancel loop baits.

Hwoarang

  • Weak Side: Sidestep Right
  • Punish: RFF D/B+4 (-13), LFS 3 (-10)
  • Strategy: Interrupt stance entry with jabs. B+1 to CH step cancels.

Jun Kazama

  • Weak Side: Sidestep Right

Alisa Bosconovitch

King

  • Strategy: Delay string to bait throw mix. Backdash out of chain throw range.

10. Anti-Steve Tech

This section explores how experienced opponents counter Steve’s stance system and how to adapt to reverse these counters effectively.

Common Steve Counters

  • High Crush: Duck under FLK 1 or jab strings
  • Sidewalk Right: Evades FLK 2, FLK 1+2
  • Low Parry: Common vs D+1 or Ducking lows
  • Throw Tech: PAB 1+2 is predictable when overused

Parry and Sabaki Risk

  • Jun’s Parry: Hits FLK 2, PAB 1
  • Leroy Hermit Sabaki: Breaks FLK 1+2 strings
  • Asuka’s Reversal: Stops linear cancel attempts

Stance Break Chart

StanceOpponent CounterHow to Beat It
FLKSidewalk RightCancel → DCK 1 or throw
PABJab interruptDelay → B+1 (CH trap)
DCKLow parryCancel → FLK 2 → FLK 1+2
SwayBackdash or waitEmpty cancel → jab → trap

Counter-Adaptive Strategy

  • If opponent sidewalk FLK: Cancel → DCK 1+2 or FLK 1
  • If opponent fuzzy guards PAB: Delay jab into throw/low
  • If opponent low parries DCK: Cancel → jab → sway reset
  • If opponent parries FLK: Use CH jab loop → backdash FLK 2

11. Controller Tips

Optimize your input setup whether you play on pad or stick. Includes layout options, input drills, and motion training for Steve’s cancel-heavy playstyle.

Recommended Layouts

  • Pad: Assign 1+2 and 3+4 to shoulder buttons (L1/L2)
  • Stick: Map DCK 1+2 or FLK 1+2 to LP+RP macro
  • Optional: Heat Burst on R2/RT for fast conversion

Execution Drills

  • FLK 1 → cancel → jab → DCK 1 (repeated)
  • 2,1F → PAB → sway → FLK 1+2 loop
  • Heat Dash → DCK 1 → FLK jab → FLK 2

Motion Training

  • Backdash → FLK cancel → sway → reset position
  • WS 1 → DCK 1+2 → FLK jab trap
  • F/F+2 → Heat Dash cancel → FLK stance entry

Consistency Checklist

  • Practice FLK cancel loops 50x daily
  • Heat conversion drills on dummy side switch
  • Record replays and watch input timing errors
